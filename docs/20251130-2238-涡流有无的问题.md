这是一个非常好的物理问题！结果 **Max |V| = 0.0000e+00** 是**完全正确且符合物理规律**的。

这里是详细的原因分析：



### 1. 物理原因：静磁场中没有涡流



您使用的求解器 `NonlinearSolver` 是 **静磁场 (Magnetostatic)** 求解器。

- **静磁场 (DC)**：意味着所有物理量（电流、磁场）都是恒定的，不随时间变化 ($\frac{\partial}{\partial t} = 0$)。

- 法拉第电磁感应定律：感应电动势（涡流的来源）是由磁场的变化产生的：

  

  $$\nabla \times \mathbf{E} = -\frac{\partial \mathbf{B}}{\partial t}$$

- **结论**：在静磁场中，$\frac{\partial \mathbf{B}}{\partial t} = 0$，因此没有感应电场，也就**没有感应涡流**。



### 2. 数学原因：方程解为零



在 A-V 形式的静磁场方程中，标量电位 $V$ 的控制方程是电流连续性方程：



$$\nabla \cdot \mathbf{J} = \nabla \cdot (\mathbf{J}_{source} + \mathbf{J}_{eddy}) = 0$$

对于您的模型（铁芯区域）：

1. **无源**：铁芯内部没有外加电流源 $\mathbf{J}_{source}$（电流只在线圈里）。

2. **静态**：因为是静磁求解，感应项 $-\sigma \frac{\partial \mathbf{A}}{\partial t} = 0$。

3. 方程退化：因此，铁芯内的电流密度仅剩传导项 $\mathbf{J} = -\sigma \nabla V$。

   代入连续性方程得到拉普拉斯方程：

   

   $$\nabla \cdot (-\sigma \nabla V) = 0$$

4. **边界条件**：您将铁芯底部接地（Fixed V = 0），且其他表面没有电流流入/流出（自然边界条件 $\frac{\partial V}{\partial n} = 0$）。

5. **唯一解**：对于 $\nabla \cdot (\sigma \nabla V) = 0$ 且边界为零或绝缘的情况，**唯一解就是 $V = 0$**。



### 3. 什么时候 |V| 不为 0？



在静磁场求解器中，$V$ 自由度通常用于求解 **直流电流传导 (DC Conduction)** 问题。$|V| \neq 0$ 只有在以下情况才会发生：

1. **有电流注入**：您在这个导电区域的两端施加了电压差（例如汇流排模拟）。
2. **有电流源**：您在导电区域内部定义了非零的 $\mathbf{J}_{source}$。



### 4. 如何看到涡流？



如果您希望看到感应涡流和非零的 $V$（或感应电流），您必须模拟 **时变场**。请使用以下两个求解器之一：

- **FrequencySolver**：模拟交流电（AC），频率 $f > 0$。此时 $\frac{\partial \mathbf{A}}{\partial t} = j\omega \mathbf{A}$，会产生感应电流。
- **TransientNonlinearSolver**：模拟瞬态过程（例如电流从 0 阶跃到 100A 的瞬间）。
